# Fix Report: 會員語言切換 + 桌邊服務併入後台內用訂餐

## 已完成

### 1. 會員登入後新增中英文切換
- 在會員控制台右上角新增 `EN / 中` 語言切換按鈕。
- 切換語言後，會員控制台的四個功能頁籤會同步更新：
  - 當前訂單 / Current Orders
  - 桌邊服務 / Table Service
  - 歷史紀錄 / Order History
  - 個人資料修改 / Profile
- 桌邊服務內的標題、說明、座位選擇按鈕、服務項目也會跟著切換中英文。

### 2. 桌邊服務即時進入後台「內用訂餐」區
- 會員在桌邊服務選擇座位並送出服務後，後台「購物車 / 訂單資訊」裡的左側「內用訂餐」會立即顯示新的服務任務框。
- 服務任務框樣式沿用訂單方框，未處理時為紅色。
- 顯示內容包含：
  - 桌號
  - 服務內容
  - 數量
  - 會員姓名
  - 電話
  - 建立時間

### 3. 完成後刪除
- 後台在「內用訂餐」區的服務任務框按下「完成」後，會刪除該桌邊服務。
- 同時也會同步更新原本的「桌邊服務」分頁資料。

### 4. 後台刷新
- 後台每 4 秒會重新整理訂單與桌邊服務。
- 手動按「重新整理」也會同步刷新訂單與桌邊服務。

## 測試結果
- `npm run build` 成功。
- `server.js` 語法檢查通過。
- 桌邊服務 API POST 測試成功。
- 後台桌邊服務 API GET 測試成功。
